# 实验三
## 1.实验目的：
- 输入是两个Word文档，一个是原始试题，另一个是标准答题。要求编写程序根据这两个文件，自动输出出题的题目。
- 输出是一道题目，以及题目的总分。根据试题和标准答案的各种情况不同，可能有以下3种不同的输出：
```
替换题：请将文中所有的文字“***”替换为“***”。 总分：**分
替换题：请删除文中所有的文字“***”。总分：**分
没有替换题！
其中的总分表示替换出现的次数
```
